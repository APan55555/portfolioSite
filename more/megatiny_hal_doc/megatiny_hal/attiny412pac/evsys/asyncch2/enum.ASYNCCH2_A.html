Asynchronous Channel 2 Generator Selection

Value on reset: 0

Variants

OFF

0: Off

CCL_LUT0

1: Configurable Custom Logic LUT0

CCL_LUT1

2: Configurable Custom Logic LUT1

AC0_OUT

3: Analog Comparator 0 out

TCD0_CMPBCLR

4: Timer/Counter D0 compare B clear

TCD0_CMPASET

5: Timer/Counter D0 compare A set

TCD0_CMPBSET

6: Timer/Counter D0 compare B set

TCD0_PROGEV

7: Timer/Counter D0 program event
